# opnsense-shell-utils

A collection of shell utilities for OPNsense administration.

## Utilities

| Utility | Description |
|---------|-------------|
| [generate-ovpn](./generate-ovpn/) | Generate `.ovpn` client configs from OPNsense config.xml, with support for multiple CA certificates |

## Requirements

- OPNsense (FreeBSD-based)
- `xmllint`
- `base64`
